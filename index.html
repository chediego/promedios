Instrucciones

-en caso de que todas las notas tengan igual ponderacion, solo rellenar el campo nota.
-en caso que cada nota tenga distinta ponderacion, debera rellenar el campo porcentaje tambien
ej:
nota: 4.2 porcentaje : 40
nota2: 5.5 porcentaje2 : 60
-en caso que desee saber que nota(s) nesecita para pasar
el ramo, con distintas ponderaciones, debera rellenar el campo
porcentajes y/o el campo notas, dejando vacio las notas que desee
que le calculemos(el calculo se hace para pasar el ramo con nota 4)
